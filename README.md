# TaskForge

Monorepo (frontend + backend + infra) for project/task manager.

## Structure
- frontend/ — React + TS (Vite)
- backend/ — Node/Express + TS
- infra/ — IaC Templates (later SAM/Serverless/CDK)

## Scripts
- npm run dev — Runs front and backend in parallel
- npm run build — Build both packages
